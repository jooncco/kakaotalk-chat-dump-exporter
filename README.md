## Getting Started

### 1. Install Python

https://www.python.org/downloads/

```zsh
python --version
```
```zsh
pip --version
```

### 2. Install Dependencies

```zsh
pip install opencv-python pyperclip pyautogui pillow
```

### 3. Open Kakaotalk Chat Window
It should be always open in order this application to work.

### 4. Run

```zsh
python main.py
```
